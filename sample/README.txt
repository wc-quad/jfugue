This project is a sample project aimed at becoming familiar with using the jfugue library. The base project was built with maven using the command:

	mvn archetype:generate -DgroupId=com.jfugue.sample -DartifactId=sample -DartifactArtifactId=maven-archetype-quickstart -DinteractiveMode=false

Details on this command can be found at: http://maven.apache.org/guides/getting-started/maven-in-five-minutes.html
